# AI Surveillance System

An AI-powered human detection system for CCTV, webcam, and video feeds — built as a college project to bring real-time, offline-capable surveillance intelligence to standard camera setups.

## 🔍 Overview

This system uses **YOLOv8** to detect humans in live or recorded video, drawing real-time bounding boxes around detected individuals. It comes with an interactive dashboard for monitoring feeds and supports multiple camera sources simultaneously — all without needing an internet connection.

## ✨ Features

- 🎯 **Real-time human detection** using YOLOv8
- 📹 **Multiple input sources** — CCTV (RTSP), webcam, or uploaded video files
- 🖥️ **Interactive dashboard UI** for live monitoring
- 🔁 **Multi-camera support** — monitor several feeds at once
- 📡 **Offline capability** — runs entirely without internet access
- ⚡ **Optimized for real-time performance**

## 🛠️ Tech Stack

- **Backend:** Python
- **Detection Model:** YOLOv8
- **Input:** CCTV / Webcam / Video Upload

## 📌 Status

This is currently a **working prototype**, originally developed as a college project. Actively evolving — contributions and feedback welcome!
